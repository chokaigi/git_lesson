# WindowsとMacのキーボードショートカット対応表
|            | Windowsでの操作            | Macでの操作               |
|:-----------|:--------------------------|:------------------------|
| コピー      | Ctrl+C                    | Command+C               |
| 切り取り    | Ctrl+X                    | Command+X               |
| 貼り付け    | Ctrl+V                    | Command+V               |
| 上書き保存  | Ctrl+S                    | Command+S               |
| すべてを選択 | Ctrl+A                   | Command+A               |
| 入力切替    | 半角/全角を押すたびに切り替え | 英数とかなのキーが個別に存在 |
